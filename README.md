# Pokemon Towns Adventure

A retro-style Pokemon browser game built with HTML, CSS, and JavaScript.

## Features

- Explore multiple towns
- Battle wild Pokemon
- Gym Leader battles
- Pokedex system
- Save & Load progress
- Leveling and Evolution
- Pixel-art inspired UI

## Controls

- 1 = Thunder Shock
- 2 = Quick Attack
- 3 = Tail Whip
- 4 = Potion
- ESC = Flee Battle

## Play

Open `index.html` in your browser.

## Technologies

- HTML5
- CSS3
- JavaScript
